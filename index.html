Commands are still in development and may have bugs.

          

          
spawn <selection>

          
kill <selection>

          
revive <selection>

          
heal <amount> <selection>

          
harm <amount> <selection>

          
remove <selection>

          
set <variable> <value>

          

          

          
Valid values for amount:

          Any integer value (e.g: 100)
          

          
Valid values for selection:

          <count> <type>
          "self"
          "all"
          

          
Valid values for count:

          Any integer value (e.g: 100)
          "all"
          

          
Valid values for type:

          "enemy"
          "dummy"
          "all"
          

          
Valid value for variable:

          n/a
          

          
Note: Some commands will not accept all possible values for an argument. e.g.: "spawn self" will not execute, even though "self" is listed as a valid value for a selection.
