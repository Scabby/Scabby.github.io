Note that the following list is not correct or complete.

          

          
spawn <selection>

          
kill <selection>

          
revive <selection>

          
heal <amount> <selection>

          
harm <amount> <selection>

          
remove <selection>

          
set <variable> <value> <selection>

          

          

          
Valid values for amount:

          
    Any integer value (e.g: 100)

          

          
Valid values for selection:

          
    <count> <type>

          
    "self"

          
    "all"

          

          
Valid values for variable:

          

          

          
Valid values for count:

          
    "all"

          
    Any integer value (e.g: 100)

          

          
Valid values for type:

          
    "all"

          
    "enemy"

          
    "dummy"
